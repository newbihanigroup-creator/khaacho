## Simple, Trust-First Design for Emerging Markets

---

## Design Principles

### 1. **Extreme Simplicity**
- One action per screen
- Large touch targets (minimum 48x48px)
- Clear visual hierarchy
- Minimal text, maximum icons

### 2. **Trust-First**
- Show real photos of vendors
- Display verification badges
- Show order history prominently
- Transparent pricing

### 3. **Mobile-First**
- Design for 4-5 inch screens
- Thumb-friendly navigation
- Offline-capable
- Low data usage

### 4. **Local Context**
- Support local languages (Nepali, Hindi)
- Local currency (NPR)
- Familiar payment methods
- Cultural color preferences

### 5. **Progressive Disclosure**
- Show only what's needed now
- Hide complexity
- Guide users step-by-step
- Provide help at every step

---

## Color Palette

```
Primary Colors:
- Trust Blue: #2563EB (buttons, links)
- Success Green: #10B981 (confirmations, verified)
- Warning Orange: #F59E0B (pending, attention)
- Error Red: #EF4444 (errors, urgent)

Neutral Colors:
- Dark Gray: #1F2937 (text)
- Medium Gray: #6B7280 (secondary text)
- Light Gray: #F3F4F6 (backgrounds)
- White: #FFFFFF (cards, surfaces)

Accent Colors:
- Gold: #F59E0B (premium, featured)
- Purple: #8B5CF6 (promotions)
```

---

## Typography

```
Primary Font: Inter (clean, readable)
Fallback: System fonts

Sizes:
- Heading 1: 24px, Bold
- Heading 2: 20px, Semibold
- Body: 16px, Regular
- Small: 14px, Regular
- Tiny: 12px, Regular

Line Height: 1.5x font size
Letter Spacing: Normal
```
